## 🌟 Overview

Welcome to my personal portfolio! This project is a modern, responsive, and high-performance showcase of my skills, experience, and projects. Built with **React**, **Vite**, and **Tailwind CSS**, it features a sleek design and smooth interactions.

## ✨ Key Features

- **📱 Fully Responsive Design**: Optimized for all screen sizes, from mobile to desktop.
- **🚀 High Performance**: Lightning-fast load times thanks to Vite.
- **🛠️ Dynamic Content**: Sections for Experience, Education, Projects, and Certifications.
- **📧 Contact Form**: Integrated contact section to get in touch easily.
- **🎨 Modern UI/UX**: Clean aesthetics with Tailwind CSS and Lucide icons.

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/prerana-chavan/Prerana_Portfolio.git
cd Prerana_Portfolio/Prerana/Prerana
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```
